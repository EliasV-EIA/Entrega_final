Sistema De Compras:

Este programa permite:
-Administrar productos y usuarios
-Agregar productos al carrito de un cliente

Operacion:
El sistema se opera completamente a traves de la interfaz de menus de JOptionPane. Esto es o una entrada de texto, o un menu simple.

Persistencia: 
La persistencia funciona con una base de datos local de mysql. El programa funcionara siempre en cuando esta exista en la direccion especificada
